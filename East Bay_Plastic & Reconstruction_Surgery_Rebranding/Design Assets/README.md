# Design Assets

## Overview
This folder contains **visual and UX design assets** developed to support the website redesign and brand repositioning effort.

Design work focused on clarity, trust, and usability for both **reconstructive** and **cosmetic** patient journeys.

---

## Contents
- **Logo Designs/**
  - Brand identity exploration and refinements

- **Mockups/**
  - High-level visual concepts and layout explorations

- **Color Palette & Font Style.fig**
  - Brand style guide defining typography and color usage

- **Wireframes (Desktop & Mobile)**
  - Page-level layout and user flow planning
  - Separate designs for desktop and mobile experiences

- **Visual Sitemap**
  - High-level representation of site structure and navigation

---

## Design Principles
- Mobile-first usability
- Clear service segmentation
- Trust-building visual hierarchy
- Accessibility and readability for medical audiences

---

## Outcome
Design assets informed:
- Homepage and service page layouts
- CMS implementation requirements
- Content placement and navigation strategy
